Demo - Paste & print SVGs on a letter page

Paste SVGs into the page, reposition them, then print the page. Most browsers will let you save a PDF document from the print dialog, which you can then print later.



This is a very rough work in progress! There are a bunch of basic things that aren't implemented yet:

  • Select a single item
  • Delete selected item
  • Drag-and-drop SVG files
  • Zoom and pan document, in container
  • Initial auto-zoom to fit document in viewport
  • Z-index ordering based on recent selection
  • Select multiple items
  • Delete multiple items
  • Export to SVG
